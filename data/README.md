# Epicue Raw Data & Evaluation Metrics

This directory contains the raw datasets and `snforge` simulation logs utilized for the evaluation and quantitative analysis.


## How These Files Were Created

The evaluation artifacts in this directory were generated exclusively through local Cairo VM testing to validate the mathematical boundaries of the Epicue framework.

1. **`snforge_logs/gas_metrics.json`**: Generated by running the official Starknet Foundry (`snforge test`) test suite locally against the core Cairo smart contracts. The values represent the exact L2 gas units consumed by operations compiled to Sierra and CASM bytecode during execution.
2. **`reputation_decay.csv`**: Captured from the state transition logs outputted during a chronological `snforge` simulation that advanced block timestamps to simulate decay arcs and Auditor slashing events against the Nabla-based trust gradient equations.

## Artifact Listing

- `snforge_logs/gas_metrics.json`: Base L2 computational overhead for all core governance loops.
- `reputation_decay.csv`: Tabular data mapping institutional reputation dynamics and BFT resilience.
